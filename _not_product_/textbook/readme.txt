4-2-3 図版にキャプションを挿入する（P104）
P104のブラウザ出力結果[portfolio/01.html]の表示と実際ブラウザ表示が異なり、画像（figureタグ）の上下左右に余白があります。
これは、3-5-4 normalize.cssを読み込む（P083）にて、normalize.cssが書籍執筆時よりバージョンアップ（v3）して、figureタグのマージン設定が変わったことによるものです。
書籍の表示に合わせるには下記のように図版のマージンを指定してください。

[common/css/style.css]
/* 図版 */
figure {
	margin: 0;
}

8-1-3 jQueryプラグインでスライドショーを作る
手順2 jQueryをダウンロードする（P278）

jQueryが書籍執筆時よりバージョンアップしたため、
最新バージョンのjQuery（1.11.0）では、手順4で登場するプラグイン「bxSlider」の挙動が安定せず、
スライドショーの2枚目で突っかかるようになってしまいました。

P278の手順2の右図ページ（http://jquery.com/download/）にて、下にスクロールすると、
「Past Releases」と表示されており、過去のバージョンのjQueryがダウンロードできるようになっています。
「bxSlider」の動作確認ができているバージョン「1.10.1」をダウンロードしてご利用ください。
minified・Uncompressedのどちらをダウンロードしてもかまいません。
